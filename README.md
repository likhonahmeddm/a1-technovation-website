# A1 Technovation Website

A modern static marketing website for A1 Technovation, covering web design, white-hat SEO, social media marketing, portfolio, blog, and contact pages.

```text

|-- index.html

`-- README.md
```

## Current Direction

- Premium, agency-inspired visual system
- Editorial section flow with stronger spacing and typography
- Gradient-heavy modern UI with glassmorphism accents
- Consistent brand presentation across all pages
- Dhaka-based business details applied across the site

## Main Features

- Fully static HTML, CSS, and JavaScript website
- Responsive navigation and mobile menu
- Homepage with expanded premium sections
- Dedicated service pages for SEO, web design, and social media
- Portfolio filtering
- Contact form validation
- Scroll and interaction effects

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- Font Awesome
- Google Fonts
